background-size

  • auto 默认值

    背景图像的真实大小。

  • cover

    将背景图像等比缩放或放大到完全覆盖容器，背景图像有可能超出容器。

  • contain

    将背景图像等比缩放到宽度或高度与容器的宽度或高度相等，背景图像始终被包含在容器内。

  • length

    用长度值指定背景图像大小。不允许负值

  • percentage

    用百分比指定背景图像大小。不允许负值。百分比的值相对于容器的宽高

background-clip

  • border-box 默认值

    从border区域（不含border）开始向外裁剪背景

  • padding-box

    从padding区域（不含padding）开始向外裁剪背景。

  • content-box

    从content区域开始向外裁剪背景。

  • 遮罩效果

background-origin

  • border-box

    从border区域（含border）开始显示背景图像

  • padding-box

    从padding区域（含padding）开始显示背景图像。

  • content-box

    从content区域开始显示背景图像。

background-position

  • percentage

    用百分比指定背景图像填充的位置。可以为负值

  • length

    用长度值指定背景图像填充的位置。可以为负值

  • 关键字

    用百分比指定背景图像填充的位置。center,left,right,top,bottom
